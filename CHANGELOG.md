# Changelog

## [Unreleased]

### Добавлено

- Колонка `is_likely_probe` в событиях: эвристика «шум сканирования / мусорный TLS» по подстрокам в сообщении; бэкфилл при миграции БД.
- Фильтр дашборда «Скрыть шум» и параметр API `exclude_probe=1` для истории, KPI, топа типов и списка целей.
- Переменные `PROBE_NOISE_MARKERS` / `PROBE_NOISE_EXTRA_MARKERS` для настройки маркеров шума.
- Уточнение шума для REALITY: `server name mismatch` с **непустым** SNI не считается шумом (ошибка клиента); **пустой** SNI после двоеточия — шум.

## [1.0.0] — 2026-05-02

Первый релиз.

### Добавлено

- Дашборд по `error.log` Xray: KPI, топ типов ошибок, тренд по времени.
- История событий в SQLite с пагинацией и фильтрами: тип ошибки и inbound/цель (мультивыбор, поиск в выпадающих списках), IP источника (поиск по подстроке).
- Детализация по типу ошибки (боковая панель, сырая строка, копирование).
- Обогащение IP (location/owner/asn) с кэшем и дневным лимитом внешних запросов.
- Фоновый инжест лога (APScheduler), ротация `monitor_job.log`.
- Вход через `/login` (сессия Flask), блокировка после неудачных попыток с одного IP, опционально `X-Forwarded-For` за прокси.
- Favicon и логотип на странице входа (`static/favicon.svg`).
- Пример окружения `.env.example`, документация в README.

### Технические детали

- Python 3.10+, Flask, SQLite, Chart.js + Tailwind (CDN).
- Загрузка `.env` с `override=True`, чтобы значения из файла перекрывали переменные окружения ОС.
